# 미니멀 대시보드 디자인 가이드

## 개요
특허 AI 플랫폼의 대시보드를 미니멀하고 깔끔하면서 고급스러운 디자인으로 전면 재설계하는 가이드입니다.
현대적인 UI/UX 원칙을 적용하여 사용자 경험을 극대화합니다.

## 디자인 원칙

### 1. 미니멀리즘 (Minimalism)
- **여백 활용**: 충분한 여백으로 시각적 휴식 공간 제공
- **요소 최소화**: 핵심 정보만 표시하여 인지 부하 감소
- **단순한 구조**: 직관적인 정보 계층 구조

### 2. 고급스러움 (Luxury)
- **프리미엄 색상**: 세련된 색상 팔레트 사용
- **고품질 타이포그래피**: 가독성과 미적 완성도 고려
- **정교한 애니메이션**: 부드럽고 자연스러운 전환 효과

### 3. 기능성 (Functionality)
- **정보 우선순위**: 중요한 데이터를 시각적으로 강조
- **직관적 네비게이션**: 사용자 행동 패턴 기반 설계
- **반응형 디자인**: 모든 디바이스에서 최적화된 경험

## 컴포넌트 디자인

### 1. 통계 카드 (Stats Card)
```tsx
interface StatsCardProps {
  title: string;
  value: string | number;
  change?: {
    value: number;
    type: 'increase' | 'decrease' | 'neutral';
  };
  icon?: React.ReactNode;
}

const StatsCard: React.FC<StatsCardProps> = ({ title, value, change, icon }) => (
  <div className="bg-white rounded-xl shadow-sm border border-neutral-200 p-6 hover:shadow-md transition-shadow duration-200">
    <div className="flex items-center justify-between">
      <div className="flex items-center space-x-3">
        {icon && (
          <div className="p-2 bg-accent-blue/10 rounded-lg">
            {icon}
          </div>
        )}
        <div>
          <p className="text-sm font-medium text-neutral-600">{title}</p>
          <p className="text-2xl font-semibold text-neutral-900">{value}</p>
        </div>
      </div>
      {change && (
        <div className={`flex items-center space-x-1 px-2 py-1 rounded-full text-xs font-medium ${
          change.type === 'increase' ? 'bg-green-100 text-green-800' :
          change.type === 'decrease' ? 'bg-red-100 text-red-800' :
          'bg-neutral-100 text-neutral-800'
        }`}>
          <span>{change.value > 0 ? '+' : ''}{change.value}%</span>
        </div>
      )}
    </div>
  </div>
);
```

### 3. 활동 피드 (Activity Feed)
```tsx
interface ActivityItem {
  id: string;
  type: 'search' | 'report' | 'login';
  title: string;
  description?: string;
  timestamp: string;
  status?: 'success' | 'pending' | 'error';
}

const ActivityFeed: React.FC<{ activities: ActivityItem[] }> = ({ activities }) => (
  <div className="bg-white rounded-xl shadow-sm border border-neutral-200 p-6">
    <h3 className="text-lg font-semibold text-neutral-900 mb-4">최근 활동</h3>
    <div className="space-y-4 max-h-96 overflow-y-auto">
      {activities.map((activity) => (
        <div key={activity.id} className="flex items-start space-x-3 p-3 rounded-lg hover:bg-neutral-50 transition-colors">
          <div className={`w-2 h-2 rounded-full mt-2 ${
            activity.status === 'success' ? 'bg-green-500' :
            activity.status === 'pending' ? 'bg-amber-500' :
            activity.status === 'error' ? 'bg-red-500' :
            'bg-neutral-400'
          }`} />
          <div className="flex-1 min-w-0">
            <p className="text-sm font-medium text-neutral-900">{activity.title}</p>
            {activity.description && (
              <p className="text-xs text-neutral-600 mt-1">{activity.description}</p>
            )}
            <p className="text-xs text-neutral-500 mt-1">{activity.timestamp}</p>
          </div>
        </div>
      ))}
    </div>
  </div>
);
```
